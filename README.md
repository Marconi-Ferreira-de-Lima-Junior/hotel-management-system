# 🏨 Hotel Management System

Sistema completo de gerenciamento de hotel desenvolvido com Python e Django, com foco em controle operacional, organização de reservas e gestão financeira.

---

## 🚀 Sobre o Projeto

O **Hotel Management System** é uma aplicação web que simula um sistema real de gestão de hotel, permitindo o controle de clientes, quartos, reservas e fluxo financeiro.

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento web com Django e simular um cenário próximo ao mercado.

---

## ⚙️ Funcionalidades

* 👤 Cadastro e gerenciamento de clientes
* 🛏️ Controle de quartos e disponibilidade
* 📅 Sistema de reservas com validação de datas
* 💰 Registro de receitas e despesas
* 📊 Dashboard com visão geral do sistema
* 🔐 Sistema de autenticação (login e registro de usuários)
* 📝 Relatórios básicos do sistema

---

## 🛠️ Tecnologias Utilizadas

* Python
* Django
* SQLite
* HTML5
* CSS3

---

## 📁 Estrutura do Projeto

Organizado seguindo boas práticas do Django:

* `models.py` → estrutura dos dados
* `views.py` → lógica do sistema
* `forms.py` → manipulação de formulários
* `templates/` → interface do usuário
* `static/` → arquivos CSS

## ▶️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Marconi-Ferreira-de-Lima-Junior/hotel-management-system
```

### 2. Acesse a pasta do projeto

```bash
cd Hotel_pdv
```

### 3. Crie um ambiente virtual

```bash
python -m venv venv
```

### 4. Ative o ambiente virtual

**Windows:**

```bash
venv\Scripts\activate
```

### 5. Instale as dependências

```bash
pip install -r requirements.txt
```

### 6. Execute as migrações

```bash
python manage.py migrate
```

### 7. Inicie o servidor

```bash
python manage.py runserver
```
